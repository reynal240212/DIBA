#🏆 DIBA FBC - Gestión y Administración del Club

Bienvenido a DIBA FBC, la plataforma digital diseñada para la gestión y administración del club de fútbol amateur con sede en Barranquilla, Colombia. Nuestro sitio permite a jugadores, entrenadores y administradores acceder y gestionar información clave del equipo de manera eficiente.

✨ ¿Qué encontrarás en nuestro sitio?
📋 Administración de jugadores: Registro, seguimiento y actualización de datos personales y deportivos.
⚽ Historial y trayectoria: Consulta de rendimiento, estadísticas y trayectoria de cada jugador.
📁 Gestión de documentos: Subida y almacenamiento seguro de documentos relevantes.
🔍 Ficha técnica: Información detallada sobre cada jugador, incluyendo posiciones, habilidades y más.
🌐 Accesibilidad total: Diseñado para que los niños del club y sus familias puedan usarlo desde casa.
💻 Desarrollado con ASP.NET Core, HTML, CSS y JavaScript, utilizando Supabase como base de datos y alojado en GitHub.

Este sitio busca mejorar la organización del club y brindar a los jugadores una plataforma moderna para su desarrollo deportivo. 🚀⚽
